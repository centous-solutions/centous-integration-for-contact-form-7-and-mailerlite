Seamlessly integrate MailerLite with Contact Form 7 to add subscribers directly from WordPress.

== Description ==

MailerLite Integration for Contact Form 7 allows you to automatically add form submissions from Contact Form 7 to your MailerLite subscriber lists. This plugin simplifies your email marketing process by ensuring that every submission becomes a potential subscriber, helping you grow your audience effortlessly.

== Supported Contact Forms ==

* [Contact Form 7](https://wordpress.org/plugins/contact-form-7/)

== Plugin Features ==

* Automatically add new subscribers to MailerLite upon form submission.
* Configure MailerLite API settings directly from the plugin settings page.
* Customize the contact form fields that map to MailerLite fields subscriber attributes.

== Why We Built This Plugin ==

MailerLite is a powerful email marketing tool that can help businesses effectively communicate with their audience. By integrating it with Contact Form 7, we wanted to make it easier for users to grow their mailing lists and manage their subscriber data without any manual effort.

== GDPR Compliance ==

* This plugin does not store any personal data and complies with GDPR regulations by integrating with MailerLite, which manages user data according to their privacy policy.

== Frequently Asked Questions ==

= How does this plugin integrate with MailerLite? = 

The plugin connects to your MailerLite account using API credentials you provide in the settings. Each time a form is submitted, the data is sent to MailerLite, adding the subscriber to your chosen list.

= Can I control which fields are sent to MailerLite? = 

Yes, the plugin allows you to customize which form fields correspond to MailerLite subscriber attributes. You can easily map your form fields to the appropriate MailerLite fields in the settings.

= What if a submission fails to sync with MailerLite? = 

The plugin provides sync status notifications for each submission. If a submission fails, you can check the error logs for troubleshooting tips.
